Problem:
169. Majority Element

Given an array nums of size n, return the majority element.
The majority element is the element that appears more than ⌊n / 2⌋ times. 
You may assume that the majority element always exists in the array.

Example 1:

Input: nums = [3,2,3]
Output: 3

Difficulty:
Easy

Approach:
Cancelling method. A number was selected than was checked. The number's counter was increased,
if the same element was present as next element. The counter was decreased when the next element 
was same. If the counter reached zero than the next element was used for counting and so on.

Time Complexity:
O(n)

Space Complexity:
O(1)

Solution

Select an element, than increase counter if the next element is same, 
and decrease the counter if the next element is different. When the counter becomes zero,
select the very next element as new element.
