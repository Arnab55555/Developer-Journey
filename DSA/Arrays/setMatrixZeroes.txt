Problem:
Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's.

You must do it in place.

Example 1:


Input: matrix = [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
Example 2:


Input: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
Output: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]

Approach:
Use the first row and first column of the matrix as markers instead of using extra arrays.

Traverse the matrix once.
Whenever a 0 is found, mark its corresponding row and column by setting the first element of that row and the first element of that column to 0.
Use an additional variable to keep track of whether the first column should be zeroed, since matrix[0][0] cannot represent both the first row and the first column.
Traverse the remaining matrix using the markers and set the required cells to 0.
Finally, update the first row and first column based on their respective markers.

Time Complexity:
O(m x n)

Space Complexity:
O(1)

What did I learn?
The input matrix itself can be used as auxiliary storage to achieve O(1) extra space.
The first row and first column can act as marker arrays.
A single cell (matrix[0][0]) cannot independently represent both the first row and the first column, 
so an additional variable is needed.
Tracing small examples on paper helps identify indexing and marker mistakes.

Solution

class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int rows = matrix.size();
        int cols = matrix[0].size();

        int col0 = 1;

        // Step 1: Mark rows and columns
        for (int i = 0; i < rows; i++) {
            if (matrix[i][0] == 0)
                col0 = 0;

            for (int j = 1; j < cols; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        // Step 2: Update the matrix (excluding first row and first column)
        for (int i = 1; i < rows; i++) {
            for (int j = 1; j < cols; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        // Step 3: Update the first row
        if (matrix[0][0] == 0) {
            for (int j = 0; j < cols; j++) {
                matrix[0][j] = 0;
            }
        }

        // Step 4: Update the first column
        if (col0 == 0) {
            for (int i = 0; i < rows; i++) {
                matrix[i][0] = 0;
            }
        }
    }
};
